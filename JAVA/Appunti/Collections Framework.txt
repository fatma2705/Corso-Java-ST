1) ARRAYLIST :

sintassi : ArrayList < Tipo dati> nomeLista = new ArrayList<>();



 * METODI LISTE  *


- ADD(elemento)  // aggiunge elemento

- REMOVE(elemento) // rimuove elemento

- CONTAINS(elemento) // ritorna true se l'elemento esiste dentro la lista false se non 

- isEMPTY // ritorna true se la lista è vuota false se non lo è 

- SIZE() // restituisce il numero di elementi presenti dentro la lista 

- CLEAR() // svuota la lista 

- INDEXOF(elemento) // restituisce l'indice della prima occorrenza dell'elemento specificato

- GET(int indice) // restituisce l'elemento alla posizione specificata

- SET(int indice , elemento ) // sostituisce l'elemento presente alla posizione specificata con l'elemento specificato
